Задание 1:

В программе на C/C++ имеются переменные

int a, b, c, d, e, x;

Написать вставку на ассемблере, которая вычислит выражение:

x=(((a+b)*c)-d)/e

Требования и ограничения:
 Значения переменных лежат в диапазоне [1,100]
 Деление целочисленное, остаток отбрасывается

Задание 2:

В программе на С/С++ имеются переменные:

unsigned char a,b;

Программа должна запросить значения переменных у пользователя. Написать ассемблерную вставку, которая вычислит a+b и запишет в две переменные типа bool наличие переполнения для знаковых/беззнаковых чисел.

Программа должна вывести на экран:
  Результат сложения для чисел со знаком
  Результат сложения для чисел без знака
  Было ли переполнение для чисел со знаком
  Было ли переполнение для чисел без знака

Пример:

a=255
b=255

a+b=-2 (signed)
a+b=254 (unsigned)

no overflow (signed)
overflow (unsigned)

Необходимо изучить команды: jc, jnc, jo,jno

Задание 3:

В программе на С/С++ имеется строка, завершающаяся нулевым кодом, например:

char szMessage[100] = "Madam, I\'m Adam";

Необходимо написать ассемблерную вставку, которая перевернет строку.
Например строка из примера должна быть заменена на "madA m'I ,madaM".
Для решения задачи рекомендуется использовать стек.

Задание 4:

В программе на C/C++ имеется целое число.
Написать вставку на ассемблере, которая посчитает факториал данного числа.

Требования:
  Для решения данной задачи необходимо использовать стек.
  Функция для подсчёта факториала должна вызываться рекурсивно.
